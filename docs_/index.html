nanojs

A Lightweight Reactive Framework with Built-in Utilities

Overview

nanojs is a minimalistic reactive framework designed to provide developers with powerful tools for building modern web applications with ease. The framework includes features like DOM manipulation, state management, a hash-based router, a reactive data model, and support for web components, all packed into a lightweight solution.

Features

  • Reactive state management.
  • Built-in lightweight DOM manipulation utilities.
  • Hash-based routing for single-page applications.
  • Reactive data model for dynamic updates.
  • Support for web components for modular development.

Usage

To use NanoJS in your project:

  1. Install NanoJS via npm (or add it to your project).
  2. Import the framework modules as needed in your JavaScript code.
  3. Start building reactive components and managing your state effortlessly!

Example:

        
import { state, derive, tags, add, router, sleep, component } from './nanojs'
const { h1, h2, div, button, input, sup, hr, span, a, li, ul } = tags()

const App = () => div(
    Header('example'),
    div(
        h2('state'),
        div("counter: ", counter),
        h2('derived state'),
        div("square:", counter, sup(2),'=', square),
        h2('property'),
        input({type: "number", value: _ => counter.value, disabled: true}),
        h2('state-derived property'),
        div({style: () => `font-size: ${8 + counter.value}px;`}, "Hello"),
        h2('state-derived child'),
        div(counter, sup(2), () => ` = ${square.value}`),
        hr(),
        button({ onclick: () => counter.value++ }, 'inc'),
        button({ onclick: () => counter.value-- }, 'dec'),
        button({ onclick: () => counter.value = 0 }, 'reset')
    )
)
add(document.body, App())
